Each method for matching requires a different set of data. #### NHS Number Matching File should contain an **NHS number** and a **date of birth**. Participants without correct values for both of these fields will not be successfully matched. #### UHL System Number Matching File should contain a **UHL System number**. Participants without a correct value for this field will not be successfully matched.
